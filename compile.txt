## Compiling on Windows

Simple Instructions:

You need Visual Studio C++ https://www.visualstudio.com/
Install bam (see above)

1. Run SetEnvironmentVars.bat
2. Run bam
3. Run loadclone

Other Information:

SetEnvironmentVars.bat sets the paths to the Visual Studio compiler
